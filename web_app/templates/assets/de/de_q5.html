Question :5

Which of the following has fraction in ascending order?

A.   1⁄3, 2⁄5, 4⁄7, 3⁄5, 5⁄6, 6⁄7

B.   1⁄3, 2⁄5, 3⁄5, 4⁄7, 5⁄6, 6⁄7

C.   1⁄3, 2⁄5, 5⁄6, 4⁄7, 6⁄7

D.  2⁄5, 3⁄5, 1⁄3, 4⁄7 ,5⁄6, 6⁄7

Answer:a

Converting each of the given fractions into decimal form, we get :

1⁄3 = 0.33,

2⁄5, = 0.4,

4⁄7, = 0.57,

3⁄5, = 0.6,

5⁄6, = 0.82,

6⁄7 = 0.857.

Clearly, 0.33 < 0.4 < 0.57 < 0.6 < 0.82 < 0.857

So, 1⁄3 < 2⁄5 < 4⁄7 < 3⁄5 < 5⁄6 < 6⁄7 .
