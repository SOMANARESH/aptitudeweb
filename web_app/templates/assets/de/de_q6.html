Which of the following has fractions in ascending order?

A.   3⁄5, 2⁄3, 7⁄5, 9⁄11, 8⁄9

B.   3⁄5, 2⁄3, 9⁄11, 7⁄5, 8⁄9

C.   3⁄5, 2⁄3, 7⁄9, 9⁄11, 8⁄9

D.   8⁄9, 9⁄11, 7⁄9, 2⁄3, 3⁄5

E.   8⁄9, 9⁄11, 7⁄9, 3⁄5, 2⁄3.

Answer:c

Converting each of the given fractions into decimal form, we get :

2⁄3 = 0.66,

3⁄5 = 0.6,

7⁄9 = 0.77,

9⁄11 = 0.81,

8⁄9 = 0.88

Clearly, 06 < 0.66 < 0.77 < 0.81 < 0.88. So,

3⁄5 < 2⁄3 < 7⁄9 < 9⁄11 < 8⁄9
