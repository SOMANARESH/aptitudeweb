Question :9

Which part contains the fractions in ascending order?

A.   11⁄14, 16⁄19, 19⁄21

B.   16⁄19, 11⁄14, 19⁄21

C.   11⁄14, 19⁄21. 16⁄19

D.   19⁄21, 11⁄14, 16⁄19


Clearly,

11⁄14 = 0.785,

16⁄19 = 0.842,

19⁄21 = 0.904.

Now, 0.785 < 0.842 < 0.904.

So, 11⁄14 < 16⁄19 < 19⁄21
