Which of the following has fraction in If 47.2506 = 4A + 7/B + 2C + 5/D + 6E, then the value of 5A + 3B +6C + D +3E is:

A.  53.6003

B.  53.603

C.  153.6003

D.  213.0003


4A + 7⁄B + 2C + 5⁄D + 6E = 47.2506

4A + 7⁄B + 2C + 5⁄D + 6E = 40 + 7 + 0.2 + 0.05 + 0.0006

Comparing the terms on both sides, we get :

4A = 40,

7⁄B = 7,

2C = 0.2,

5⁄D = 0.05,

6E = 0.0006

or A = 10, B = 1, C = 0.1, D = 100, E = 0.0001.

Then 5A + 3B + 6C + D + 3E = (5×10) + (3×1) + (6×0.1) + 100 + (3×0.0001)

= 50 + 3 + 0.6 + 100 + 0.0003

= 153.6003.
