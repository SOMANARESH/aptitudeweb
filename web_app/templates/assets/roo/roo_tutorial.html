SQUARE ROOTS AND CUBE ROOTS.

Square Root:

  • Square root of a number is the number, product of which by itself is equal to the given number. Square root is also known as under root or 2nd root. Square root is denoted by √   .

    If x2 = y then √ y  = x.

How to find number of digits in square root of a number of having n number of digits:

  1. Make pairs of two digits each. Single digit leg, if any, would make one pair.
  2. Digits in the square root of the number will be equal to number of pairs of the square number, alternatively divide n by 2. If quotient is a natural number, quotient is the number of digits in the square root. if quotient is a mixed number, round it up to the next higher natural number.

Cube Root:

  • Cube root of a number is the number, product of which by itself and itself again is equal to the given number. In other words, cube root of a number is the number whose cube is equal to the given number itself. Cube root is denoted by : symbol.
    It is called 3rd root.

How to find number of digits in cube root of a number with n number of digits:

  1. Make pairs of three digits each. Single (or two) digits left, if any , would make one pair.
  2. Digits in the cube-root of the number will be equal to number of pairs of the cube number.

Division Method of finding square root:
Method:

  1. Make pairs of two digits each starting from right hand side.

  2. Find the largest number whose square is equal to or less than the first part from left hand side.

  3. Find the reminder and bring down the next pair.

  4. Double the root obtained and write to left of number brought down.

  5. Write one digit to the right of number of step 4.
    Multiply the number so obtained with same single digit so that the product is equal to or less than the number of step 3.

  6. Repeat the processes from 3 to 5 till all the pairs are taken.

Prime Factor Method for Finding Square Root and Cube Root:

Steps:

  1. Write prime factors of the given number.

  2. Select the factors that are in pair.

  3. Take one number from each pair and multiply then. The product of such numbers followed by the product (written under √   sign) of the numbers which are not in pair.

Square Root of 3 or 4-Digit Square Number:
Base: square of (a b) by Duplex Method is : a2 | 2 a b | b 2.
Steps:

  1. Starting from left side, divide the given number in three parts. First part contains1 digit (in case of 3-digit number) or 2 digit (in case of 4-digit number) and second and third parts contain one digit each.

  2. Find maximum value of one-digit number (say a ) whose square does not exceed first part from left-hand side.

  3. Find one-digit number (say b) so that value of (2ab) does not exceed second part.

  4. If the third part is equal to b2, then ab is square-root of the given number.


Note:
  1. Remainders, if any, in the first and second part are carried over to second and third parts respectively.

  2. Square-root of any three or four-digit number is always a two-digit number.

  3. By using this method, we can find out the square root of a perfect square number only.

Easier Method of Finding Square-Root of 3 or 4 Digit Square Number:

Method:

  1. Make pairs of two digits each starting from right hand side.

  2. Find the largest number whose square is less than or eual to the number in left-most pair. This is the tens digit of square root.

  3. Based on unit digit of the given number, guess the unit digit of the square root.

Finding Square Root by Approximation:

Method:

  • Divide the given number by a divisor around approximate square root of the number. Take average of the divisor and quotient. This is approximate square root of the number.

Finding Cube Root by Approximation:

Method:

  • Divide the given number by a divisor around approximate cube root of the number. Divide the quotient by the same divisor again. Take average of the two divisor and final quotient.

    This is approximate cube root of the number.
