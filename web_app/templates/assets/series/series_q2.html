Question :2

Given the sequence below

28, 23, 18, 13, X , 3

So the value in the place of X will be

A.   8

B.   3

C.   13

D.   18


Solution:

  • From the given sequence,

  • 23−28 = −5

  • 18−23 = −5

  • 13−18 = −5

  • Since there is constant difference between the any two successive numbers in sequence

  • So the difference between X and 13 should be −5

  • X−13 = −5

  • X = −5+13 = 8

  • 3−8 = −5
