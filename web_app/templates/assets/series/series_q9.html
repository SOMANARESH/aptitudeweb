Question :9

Complete the sequence by filling the ? position

26, 19, 17, 13, 11, ?, 8, 7

A.   9

B.   15

C.   6

D.   10


Solution:

Here we can solve this in two methods.

Method-1: By finding the differences between two successive numbers

Method-2: By Finding the differences between two alternative numbers in the sequence

Method-1:

The differences can be shown as

(19-26)    (17-19)    (13-17)    (11-13)    (x-11)    (8-x)    (7-8)

   -7            -2            -4            -2            y            z            -1

Here y = x - 11 and z = 8 - x

Let us chose x from given options

Now Let x = 9

The difference will be -7    -2    -4    -2    -1    -1    -1

From these differences you can not observe the constant differences either between two successive numbers or between differences.

Similarly if x = 15

then difference will be -7    -2    -4    -2    4    -7    -1

Here the difference is not constant either successively or alternatively.

If x = 6

difference -7    -2    -4    -2    -5    3    -1

If x = 10

difference -7    -2    -4    -2    -1    -2    -1

Here also same. There is no constant difference if you take alternative differences.

So this method is failed.

Method-2:

If you look at the alternate numbers

The difference follows some constant

17-26 =-9

11-17 = -6

8-11 = -3

Similarly,

13-19 = -6

X-13 = Y

7-Y = Z

Now let us chose value of X

If X = 9, difference is -6 -4 -2. So it followd constant difference.

If X = 15, difference is -6 2 -8. There is no constant difference

If X = 6, difference is -6 -7 1. It is also failed

If X = 10, difference is -6 -3 -3. It is also failed.

So the Answer is X = 9
