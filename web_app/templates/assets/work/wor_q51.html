If the work done by ( x - a) men in ( x + 1 ) days is to the work done by ( x + 2) men in ( x - 1) days are in the ratio 9:10, then the value of x is equal to.

A.  5

B.  6

C.  7

D.  8


if (x-1) men in done work in (x+1) days

then one man do it in (x-1)×(x+1) days

if (x+2) men do the work in (x-1) days

then 1 man can do it in (x+2)×(x-1)

ratio of days given = 9: 10

(x+1)(x-1)⁄(x+2)(x-1) = 9⁄10

(x+1)⁄(x+2)= 9⁄10

10x+10 = 9x+18

x = 18-10

x = 8.
