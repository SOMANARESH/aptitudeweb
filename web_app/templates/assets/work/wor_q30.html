Question :30

Two pipes A and B can separately fill a cistern in 60 mins and 75 mins respectively. There is a third pipe in the bottom of the cistern to empty it. If all the three pipes are simultaneously opened, then the cistern is full in 50 minutes. In how much time the third pipe alone can empty the cistern?

A.  110minutes

B.  100minutes

C.  120minutes

D.  90minutes


Formula Used:

If a man can do a piece of work in x days,

Then his work for one day = 1⁄x

one minute work of pipe A = 1⁄60

one minute work of pipe B = 1⁄75

lets assume that pipe C can empty in x minutes

then one minute work of pipe C = 1⁄x

given that if all pipes in action time taken to fill the tank = 50 minutes.

pipe A, pipe B and pipe C one minute work =

one minute work of pipe A +

one minute work of pipe B -

one minute work of pipe C

1⁄50 = 1⁄60 + 1⁄75 - 1⁄x

1⁄x = 1⁄60 + 1⁄75 - 1⁄50

1⁄x = 5+4-6⁄300

Required time = 100 minutes.
