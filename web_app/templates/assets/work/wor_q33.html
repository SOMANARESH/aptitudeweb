Question :33

A and B together can do a piece of work in T days. if a alone can do the same work in ( T + 3 ) days and B alone can do the work in ( T + 12) days, find the value of T.

A.  3

B.  6

C.  9

D.  12


Formula Used:

If a man can do a piece of work in x days,

Then his work for one day = 1⁄x

A and B can do it in = T days

A and Bs one day work = 1⁄T

A alone can do the same work in = (T+3) days

As one days work = 1⁄(T+3)

B can do the same work in = (T+12) days

Bs one day work = 1⁄(T+12)

A and B one day work =

As one day work +

Bs one day work

1⁄T= 1⁄(T+3) + 1⁄(T+12)

= (T+12) + (T+3)⁄(T+3)×(T+12)

1⁄T= 2T+15⁄ T2+ 15 T+36

T2 = 36

T = 6.
