Question :43

A, B and C together undertook a work for 550. A and B together done 7/11 of the work. Find C’s share.

A.  50

B.  350

C.  200

D.  Data Insufficient


Work done by C = 1- 7⁄11

= 4⁄11

(in 11 parts of the work A and B did 7 parts and C completed remaining 4 parts.)

C share in wages = 4⁄11 × 550

=200.
