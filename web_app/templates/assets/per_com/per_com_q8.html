Question 8 :

How many numbers of 4 digits can be formed with the digits 0, 1, 2 and 3(Repetition is not allowed)

A.192

B.108

C.18

D.48


Solution:

Let’s assume four digit number be WXYZ

From the given digits the four digit combination should have thousand’s place value other than “0” .

The combinations should be from

W X Y Z where W>0

The possibilities for first place W are {1, 2, 3 } = 3 ways.

Now given that the sequence doesn’t allow repetition.

So the remaining combination of numbers are

{0, 2, 3},{0, 1, 3},{0, 1, 2}

There are 3 positions for 3 numbers. The number of ways of arrangement of 3 numbers for 3 positions is

3 P 3 = 3! ⁄ (3 - 3)! = 3×2×1 ⁄ 0!

3 P 3 = 6

So total number of ways = 6 × 3 = 18
