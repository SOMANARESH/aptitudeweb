Formulae


Factorial:

  • Factorial of n is n! = 1×2×3× ---- (n-2)×(n-1)×n or n×(n-1)×(n-2) ------ 3×2×1
    n factorial can be written as n! = n×(n-1)!
    where (n-1)! = (n-1)×(n-2)!
    where (n-2)! = (n-2)×((n-3)! and so on
    So n! = n×(n-1)! = n×(n-1)×(n-2)! = n×(n-1)×(n-2)×(n-3)! = n×(n-1)×(n-2)×(n-3)×(n-4)× ---- 4×3×2×1

Permutations:

  • n P r = n! ⁄ (n-r)!
  • n P n = n! ⁄ (n-n)! = n! ⁄ 0! = n!
  • n P 0 = n! ⁄ (n-0)! = n! ⁄ n! = 1
  • n P n = n P (n - 1)

Combinations:

  • n C r = n! ⁄ (r! × (n-r)!)
  • n C r = n C (n - r)
  • n C 0 = 1
  • n C n = 1
  • n C r + n C r+1 = (n + 1) C r
  • n C x = n C y where y = n - x
