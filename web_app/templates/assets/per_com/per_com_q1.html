Question 1 :

In how many ways the letters of a word “NUMBER” can be formed?

A.120

B.720

C.82

D.164


Solution:

Let us take first letter N. The number of ways = { N } = 1 = 1!

Let us take first two letters. N,U

The number of ways the letters can be formed = { NU,UN } = 2 = 2!

Let us take first three letters. N,U,M

The number of ways the letters can be formed ={ NUM, NMU, UNM, UMN, MNU, MUN }=6= 3!

In the same ways the number of ways the letters can formed for n letters = n!

So Finally the number of ways of arranging the letters of the word “NUMBER” = 6! = 720


Alternative Solution:

In case of alphabets, order is considered into account.

Because NUM and MUN are different even though both contains same letters.

So the number of ways = 6 P 6 = 6! = 720
