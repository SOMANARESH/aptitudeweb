Question 16 :

In an Examination, a candidate is required to answer 7 questions in all, from 3 sections having 5 questions each. What are the total number of ways in which a candidate can select the questions, provided that at least two questions are to be attempted from each section?

A.300

B.3000

C.100

D.1000

Answer: B

Solution:

Here the total number of questions from all sections = 3(5) = 15

Given that the candidate has to answer total of 7 from all sections such that he has to answer at least two questions from each sections.

So the possibilities of choosing the questions from each section are {223, 232, 322}

here 223 means, 2 questions from 1st section, 2 questions from 2nd section and 3 questions from 3rd section.

The number of ways of selecting 2 questions from 1st section = 5 C 2

5 C 2 = 5! ⁄ 2! × (5 - 2)!

5 C 2 = 5×4×3! ⁄ 2! × 3!

5 C 2 = 5×4 ⁄ 2×1 = 10

Similarly from 2nd section = 5 C 2 = 10

From 3rd Section = 5 C 3 = 5 C 2 = 10

  • n C r = n C (n - r)

    • For the 1st combination, the number of ways = 10×10×10 = 1000

    Similarly for 2nd group = 5 C 2 × 5 C 3 × 5 C 2 = 10 × 10 × 10 = 1000

    For 3rd Group = 5 C 3 × 5 C 2 × 5 C 2 = 10×10×10 = 1000

    So total number of ways = 1000 +1000 + 1000 = 3000
