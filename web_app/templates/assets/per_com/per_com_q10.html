Question 10 :

There is a five digit code consist of letters and digit. the code can have one digit from 1,2,3 and any four letters from A,b,c,d,e. first and last code must be a letter and all letters can be repeated. How many combinations can be possible?

A.1875

B.5625

C.1125

D.11250

Answer: B

Solution:

Here the first and last digit must be letters. So those two digits must be selected from 5 letters and repetition is allowed.

Scenarios

  • Let’s take 2 numbers(1,2) two positions.

    combinations are {11, 12, 21, 22 } = 4 = 22

  • Now let's take 2(1, 2 ) numbers 3 positions.

    Combinations = {111, 112 ,121, 122, 211, 212, 221, 222} = 23

  • 3 numbers (1,2,3) 2 positions

    Combinations = {11,12,13,21,22,23,31,32,33} = 9 = 32

  • Like that 4 numbers and 3 positions

    Number of Combinations = 43 = 64

As per the above Scenarios, the number of combinations of 5 numbers for two positions will be 52= 25

Now among the 3 digits, one digit have number.

So the 3 positions may contain any of 3 numbers.

the number of combinations = 3×3 = 9

Now we have 2 digits pending which are filled by 5 numbers(Because Repetition is allowed).

From above Scenarios, the number of combinations with 5 numbers and 2 digits is 52 = 25

Total number of ways = 25 × 9 × 25 = 5625
