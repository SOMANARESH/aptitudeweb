Question 2 :

In How many ways the letters of the word “MATHEMATICS” can be formed?

A.100020

B.120960

C.2494800

D.4989600


Solution:

In a given word “MATHEMATICS” there are 11 letters

Among all there are three letters { M,A,T } occurred two times in a word.

    Scenario:

  • Let us take a word SEED which has 4 letters. Among the all one letter { E } occurred two times.

    The combinations are { DEES, DESE, DSEE, EDES, EDSE,EEDS,EESD, ESDE,ESED, SDEE, SEDE, SEED } = 12 = 4! ⁄ 2!

As per the above Scenario we can calculate the number of ways as

11! ⁄ (2! × 2! × 2!) = 39916800 ⁄ 8 = 4989600
