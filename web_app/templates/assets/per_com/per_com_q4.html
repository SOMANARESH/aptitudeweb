Question 4 :

In how many ways the letters in a word “BANGALORE” can be formed by treating all vowels as a single group(all vowels comes together).

A.15120

B.90720

C.1440

D.8640

Answer: D

Solution:

In a given word the number of letters are { A, A, B, E, G, L, N, O, R } = 9

The vowels are {A, A, E, O } = 4

So each combination should consist of a vowel group as a single letter. Let’s treat the group as “V”(Just for assumption and its not a letter “V”)

So Now we have total of 6 letters including the Group.

The number of ways = 6! = 6×5×4×3×2×1 = 720

Now come to the group {A,A,E,O}

The number of ways of forming a word from the letters of the Group


4! ⁄ 2! = 4×3×2×1 ⁄ 2×1 = 12

So the total number of ways = Number of ways by treating vowels group as a single × number of ways of forming a word within a vowel group = 720 × 12 = 8640
