Question 9 :

How many numbers of 4 digits can be formed with the digits 0, 1, 2 and 3(Repetition is allowed)

A.192

B.108

C.18

D.48


Solution:

Let’s assume four digit number be WXYZ

From the given digits the four digit combination should have thousand’s place value greater than “0”.

W X Y Z where W>0

The possibilities for first place are {1, 2, 3 } = 3 ways.

Now given that the sequence allows repetition.

There are 3 positions and 4 numbers.which means each number will come again.

Scenarios

  • Let’s take 2 numbers(1,2) two positions.

    combinations are {11, 12, 21, 22 } = 4 = 22

  • Now let's take 2(1, 2 ) numbers 3 positions.

    Combinations = {111, 112 ,121, 122, 211, 212, 221, 222} = 23

  • 3 numbers (1,2,3) 2 positions

    Combinations = {11,12,13,21,22,23,31,32,33} = 9 = 32

  • Like that 4 numbers and 3 positions

    Number of Combinations = 43 = 64

There are total of 43 combinations.

Total number of ways = 3 × 64 = 192
