If 1⁄3 of A = 75% of B = 0.6 of C, then A:B:C is:

A.  4:5:9

B.  5:9:4

C.  9:5:4

D.  9:4:5

Answer: D

1⁄3A = 75⁄100B = 0.6C

1⁄3A = 3⁄4 B = 3⁄5 C

1⁄3A = 3⁄4B

B = (4⁄(3×3))×A

B = (4⁄9)×A

1⁄3A = 3⁄5C

C = (5⁄(3×3))×A

C = (5⁄9)×A

A:B:C = A : [(4⁄9)×A] : [(5⁄9)×A]

A:B:C = A×9⁄9 : [(4⁄9)×A] : [(5⁄9)×A]

A:B:C = 9A:4A:5A

A:B:C = 9:4:5
