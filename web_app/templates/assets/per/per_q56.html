If 30% of A = 0.25 of B = 1⁄5 of C, then A:B:C is

A.  5:6:4

B.  5:24:5

C.  6:5:4

D.  10:12:15

Answer: D

Given that

30% of A = 0.25 of B = 1⁄5 of C

30⁄100 A = 25⁄100 B = 1⁄5 C

3⁄10 A = 1⁄4 B = 1⁄5 C

3⁄10A = 1⁄4B

B = ((3×4)⁄10)×A

B = (12⁄10)×A

3⁄10A = 1⁄5C

C = ((3×5)⁄10)×A

C = (15⁄10)×A

A:B:C = A : [(12⁄10)×A] : [(15⁄10)×A]

A:B:C

= [A×10⁄10] : [(12⁄10)×A] : [(15⁄10)×A]

A:B:C = 10A : 12A : 15A

A:B:C = 10:12:15
