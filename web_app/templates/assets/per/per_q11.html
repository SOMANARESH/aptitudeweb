The total number of boys in a school is 15% more than the total number of girls in the school. What is the ratio of number of boys to the number of girls in the school?

A.  13:7

B.  7:13

C.  23:20

D.  None of these

Answer: C

Let the number of boys and girls in the school be X and Y

Given that The total number of boys in a school is 15% more than the total number of girls

Given that the number of boys = Number of girls +15% of girls

X = Y+[ Y×(15⁄100) ]

X = Y+( 15Y⁄100 )

X = (100Y+15Y)⁄100

X = 115Y⁄100

X⁄Y = 23⁄20
