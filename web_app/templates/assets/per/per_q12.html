If the difference between 73% of a number and 48% of the same number is 615, what is 15% of the number?

A.  123

B.  92.75

C.  369

D.  None of these


Let the number be x

Given the difference between 73% of a number and 48% of a number is 615

Then (73-48)% of x = 615

25% of x = 615

x×25⁄100 = 615

x = 100×615⁄25

x = 61500⁄25

x = 2460

Now 15% of a number is

15% of x = (15⁄100)×2460

15% of x = 36900⁄100

15% of x = 369
