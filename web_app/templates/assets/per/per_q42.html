Question :42

605 sweets were distributed equally among children in such a way that the number of sweets received by each child is 20% of the total number of children. How many sweets did each child receive?

A.  24

B.  11

C.  45

D.  None of these


Let number of childs = x

Then each child got = 20% of x

Total Sweets distributed = 20% of x × Number of childs

20% of x×x= 605

(20⁄100)x2 = 605

(1⁄5)x2 = 605

x2 = 605×5

x2 =3025

x = 55

Each child got = 20% of 55

Each child got = (20⁄100)×55

Each child got = (1⁄5)×55

Each child got = 11
