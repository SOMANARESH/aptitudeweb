If 2⁄3 of A = 75% of B = 0.6 of C, then A:B:C is:

A.  2:3:4

B.  3:4:5

C.  4:5:6

D.  9:8:10


2⁄3A=3⁄4B = 3⁄5C

2⁄3A=3⁄4B

B = ((2×4)⁄(3×3))×A

B = (8⁄9)×A

2⁄3A = 3⁄5C

C = ((2×5)⁄(3×3))×A

C = (10⁄9)×A

A:B:C = A : [(8⁄9)×A] : [(10⁄9)×A]

A:B:C = A×9⁄9 : [(8⁄9)×A] : [(10⁄9)×A]

A:B:C = 9A:8A:10A

A:B:C = 9:8:10.
