Question :68

When 60% of a number is added to 48, the result is the number itself. What is the number?

A.  80

B.  90

C.  120

D.  None of these


Let the required number be X

Given that 48 + 60% of X = X

X-(60⁄100)X = 48

((100X-60X)⁄100) = 48

(40X⁄100) = 48

(2X⁄5)X = 48

X = 48×5⁄2

X = 120
