A number when increased by 25% gives 80. The number is:

A.  60

B.  64

C.  96

D.  100


Let the number be X

Given that Number increamented by 25% gives 80

i.e. X + 25% of X = 80

X+(25⁄100)X = 80

X((100+25)⁄100) = 80

X(125⁄100) = 80

X(5⁄4) = 80

X = 80×4⁄5

X = 320⁄5

X = 64
