If 20% of A = 30% of B = 1⁄6of C, then A:B:C is:

A.  2:3:16

B.  3:2:16

C.  10:15:18

D.  15:10:18

Answer: D

Given that

20% of A = 30% of B = 1⁄6 of C

20⁄100 A = 30⁄100 B = 1⁄6 C

20⁄100 A = 30⁄100 B

B = (20⁄30)A

B = (2⁄3)A

Now 20⁄100 A = 1⁄6 C

C = (120⁄100)A

C = (12⁄10)A

A:B:C = A : (2⁄3)A : (12⁄10)A

{ L.C.M of 3 and 10 = 30 }

A:B:C = (30⁄ 30)A : (20⁄30)A : (36⁄30)A

A:B:C = 30:20:36

A:B:C = 15:10:18
