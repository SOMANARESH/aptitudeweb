Question :89

A sum of 817 is divided among A,B and C such that A receives 25% more than B and B receives 25% less than C. what is As share in the amount?

A.  285

B.  300

C.  315

D.  None of these


Given that A+B+C = 817

Let us assume that C receives X

Then B receives 25% less than C

B = C - 25% of C

B = C - (25⁄100)C

B = ((100-25)⁄100)C

B = (75⁄100)C

B = (3⁄4)C

A receives 25% more than B

A = B + 25% of B

A = B + (25⁄100)B

A = ((100+25)⁄100)B

A = (125⁄100)B

A = (5⁄4)×(3⁄4)C

A = ((5×3)⁄(4×4))C

A = (15⁄16)C

Now A:B:C = (15⁄16)C : (3⁄4)C : C

A:B:C = (15⁄16) : (3⁄4)×(4⁄4) : (16⁄16)

A:B:C = 15:12:16

As share

= (As portion⁄Sum of all portions)×Total amount

As share = (15⁄(15+12+16))×817

As share = (15⁄43)×817

As share = (12255⁄43)

As share = 285
