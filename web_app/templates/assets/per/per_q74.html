Question :74

Given A is 50% larger than C and B is 25% larger than C, then A is what percent larger then B?

A.  20%

B.  25%

C.  50%

D.  75%

Answer: A

Given that

A = 50% more than C and B = 25% more than C

A = 50% more than C

A = C + (50⁄100)C

A = ((100C+50C)⁄100)

A = (150C⁄100)

A = (3C⁄2)

C = (2⁄3)A--->(1)

B = 25% more than C

B = C + (25⁄100)C

B = ((100C+25C)⁄100)

B = (125C⁄100)

B = (5C⁄4)

C = (4⁄5)B--->(2)

From (1) and (2)

(2⁄3)A = (4⁄5)B

A⁄B = (4×3)⁄(5×2)

A⁄B = 12⁄10

A = B×12⁄10

A = 1.2B

A = B+0.2B

A = B + 20% of B

A is 20% larger than B
