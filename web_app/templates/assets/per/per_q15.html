Question :15

If 120 is 20% of a number, then 120% of that number will be.

A.  20

B.  120

C.  480

D.  720


Let the number be X

Given that 20% of X = 120

(20⁄100)×X = 120

X = 120×100⁄20

X = 12000⁄20

X = 600

Now 120% of X =

= (120⁄100)×600

= 120×600⁄100

= 72000⁄100

= 720
