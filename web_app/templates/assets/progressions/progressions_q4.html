Question :4

Find the first number of the sequence in A.P where common difference is 13, number of terms is 17 and final term is 312

A.   100

B.   104

C.   98

D.   106


Solution:

Given difference d = 13

Total number of terms n = 17

final term = tn = 312

We know that Final term is

tn = a + (n − 1)d

312 = a + (17 − 1)13

312 = a + (16 × 13)

312 = a + 208

a = 312 − 208

a = 104
