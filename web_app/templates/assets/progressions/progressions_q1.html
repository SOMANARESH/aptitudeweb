Question :1

What is the next number of the following sequence which is in Arithmetic Progression

1, 3, 5, 7, 9, 11, ?

A.   13

B.   10

C.   8

D.   14

Answer: A

Solution:

Method 1:

From the given sequence

First Term a = 1

Now let us find the common difference d

d = difference between two successive numbers in the sequence

d = 3 − 1 = 2

d = 5 − 3 = 2

So similarly for other numbers in the sequence.

There is same difference between two successive numbers.

Then the next number of the sequence = previous number + common difference

i.e, 11 + 2 = 13

Method 2:

Given the sequence.

First term a = 1

The common difference is calculated as

d = difference between two successive numbers

d = 3 − 1 = 2

d = 5 − 3 = 2

d = 7 − 5 = 2

So the next number is 7th number of the sequence

so n = 7

As per the common difference, we can say that the sequence is in Arithmatic Progression.

The nth term of the sequence in A.P is

tn = a + (n− 1)d

so 7th term = t7 = a+ (7− 1)d

t7 = 1 + (6 × 2)

t7 = 1 + 12

t7 = 13
