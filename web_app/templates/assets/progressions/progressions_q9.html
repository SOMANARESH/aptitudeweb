If X, X3+1, X4 are in arithmatic progression then the possible sum of three terms of the arithmetic progression will be

A.   7, 4

B.   27, 0

C.   27, 4

D.   7, 0

Answer: B

Solution:

Given that X, X3+1, X4 are in arithmatic progression

So common difference between two successive numbers is same

(X3+1) − X = (X4) − (X3+1)

X3+1 − X = X4 − X3 −1

X3+1 + X3+1 − X4 − X = 0

2X3+ 2 − X4 − X = 0

2(X3+1) − X(X3+1) = 0

(X3+1)(2 − X) = 0

from above eq. we have two possibilities

2 − X = 0 = > X = 2

X3+1 = 0

X3 = −1 => X = − 1

Sum of 3 terms is

S3 = X + X3+1 + X4

If X = 2, then the sum is

S3 = 2 + 23+1 + 24

S3 = 2 + 8 + 1 + 16

S3 = 27

If X = − 1, then sum is

S3 = (− 1) + (− 1)3+1 + (− 1)4

S3 = (−1) + (− 1) + 1 + 1

S3 = 0
