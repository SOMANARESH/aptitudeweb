Question :18

If the ratio of H.M to G.M of two numbers is 12:13 then the rario of numbers is

A.   2⁄5   or   5⁄2

B.   4⁄9   or   9⁄4

C.   2⁄3   or   3⁄2

D.   3⁄4   or   4⁄3

Answer: B

Solution:

Let the two numbers be a, b

We know the H.M = (2ab)⁄(a + b)

G.M = √ (ab) 

Given the ratio of H.M to G.M is 12:13

H.M⁄G.M = 12⁄13

[(2ab)⁄(a + b)]⁄√ (ac)  = 12⁄13

(2ab)⁄[(a + b) × √ ab ] = 12⁄13

(2 √ ab )⁄(a + b) = 12⁄13

Squaring on Both Sides

4ab⁄(a + b)2 = 144 ⁄ 169

ab⁄(a2 + b2 + 2ab) = 144⁄(169 ×4)

ab⁄(a2 + b2 + 2ab) = 36⁄169

(a2 + b2 + 2ab)⁄ab = 169⁄36

[(a2 + b2)⁄ab] + 2 = 169⁄36

(a2 + b2)⁄ab = (169 ⁄ 36) − 2

(a2 + b2)⁄ab = (169 − 72)⁄36

(a2 + b2)⁄ab = (97) ⁄ 36

a2 + b2 = 97 and ab = 36

(a − b)2 = a2 + b2 − 2ab

(a − b)2 = 97 − 2(36)

(a − b)2 = 97 − 72 = 25

a − b = ±5

(a + b)2 = a2 + b2 + 2ab

(a + b)2 = 97 + 2(36)

(a + b)2 = 97 + 72

(a + b)2 = 169

a + b = ±13

Let us take positive values

a + b = 13 and a − b = 5

Adding above two

a + b + a − b = 13 − 5

2a = 8

a = 4

a + b = 13

4 + b = 13

b = 13 − 4

b = 9

The ratio of numbers is a⁄b = 4⁄9

Similarly let us take negative values

a + b = −13 and a − b = −5

Adding both Eqs.

2a = − 18

a = − 9

(−9) + b = −13

b = −13 + 9

b = −4

So ratio = a⁄b = (−9)⁄(−4)

a⁄b = 9⁄4
