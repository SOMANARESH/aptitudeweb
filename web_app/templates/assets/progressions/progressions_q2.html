Question :2

The number of terms in the following sequence is

−4 ,−2 ,0, 2, 4, 6, 8, 10,- - - - - ,200 which are in A.P

A.   100

B.   101

C.   102

D.   103


Solution:

Here Initial term a = − 4

Final term = 200

Common Difference = d

d = − 2 − (−4) = 2

d = − 2 + 4

d = 2

We know that final number = tn

tn = a + (n − 1)d

200 = − 4 + (n − 1)2

200 + 4 = (n − 1)2

204 = (n − 1)2

n − 1 = 204⁄2 = 102

n = 102 + 1

n = 103
