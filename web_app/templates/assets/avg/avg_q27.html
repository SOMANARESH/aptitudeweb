The average of three numbers is 165. The first number is twice the second and thrice the third. Find the difference between the first and the third number?

A.  60

B.  180

C.  360

D.  None of these


Given that first number is twice that of second and thrice that of third

i.e.X1 = 2(X2)

X1 = 3(X3)

So 3(X3) = 2(X2)

X2 = 3(X3)⁄2

Then average of these numbers is 165

(sum of the three numbers)⁄3 = 165

X1+X2+X3 = 3×165

3(X3)+(3(X3)⁄2)+X3 = 495

(6X3+3X3+2X3 ) ⁄2 = 495 11X3 = 990

X3 = 990⁄11

X3 = 90

So X1 = 3X3

X1 = 3(90) = 270

X2 = 3(X3)⁄2

X2 = 3(90)⁄2

X2 = 270⁄2

X2 = 135

Finally the difference between first and third number is

X1-X3 = 270-90

X1-X3 = 180
