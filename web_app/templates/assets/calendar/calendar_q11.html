Question 11 :

How many days are there in x weeks x days?

A.7 x2

B.14x

C.7

D.8x


Solution:

  • Given that there are x weeks.

  • For a week there are 7 days

  • So X weeks should have = x × 7 = 7x days

  • So X weeks and X days should have = 7x + x days

  • There are total of 8x days in X weeks and X days
