Question 9 :

The calendar for the year 2007 will be the same for the year?

A.2016

B.2017

C.2018

D.2019


Solution:

  • Count the number of odd days from the year 2007 onwards to get the sum equal to 0 odd day.

  • 2007 (Normal Year) = 1 Odd day, Count = 1, Odd days = 1%7 = 1

  • 2008 (Leap Year) = 2 Odd days, Count = 1+2 = 3,Odd days = 3%7 = 3

  • 2009 (Normal Year) = 1 Odd day, Count = 3+1 = 4,Odd days = 4%7 = 4

  • 2010 (Normal Year) = 1 Odd day, Count = 4+1 = 5,Odd days = 5%7 = 5

  • 2011 (Normal Year) = 1 Odd day, Count = 5+1 = 6,Odd days = 6%7 = 6

  • 2012 (Leap Year) = 2 Odd day, Count = 6+2 = 8,Odd days = 8%7 = 1

  • 2013 (Normal Year) = 1 Odd day, Count = 8+1 = 9,Odd days = 9%7 = 2

  • 2014 (Normal Year) = 1 Odd day, Count = 9+1 = 10,Odd days = 10%7 = 3

  • 2015 (Normal Year) = 1 Odd day, Count = 10+1 = 11,Odd days = 11%7 = 4

  • 2016 (Leap Year) = 2 Odd day, Count = 11+2 = 13,Odd days = 13%7 = 6

  • 2017 (Normal Year) = 1 Odd day, Count = 13+1 = 14,Odd days = 14%7 = 0

  • Calendar for the year 2018 will be the same as for the year 2007.
