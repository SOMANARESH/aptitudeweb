If (5x2 − 3y2) : xy = 11:2, then the positive value of x⁄y is:

A.  7:2

B.  5:2

C.  3:2

D.  5:3


(5x2 − 3y2)÷(xy) =11⁄2

2(5x2- 3y2)= 11xy

10x2 − 11xy- 6y2 = 0

10x2 − 15 xy + 4xy -6y2 = 0

5x(2x-3y)+2y(2x-3y)=0

(5x+2y)(2x-3y)=0

5x= −2y and 2x = 3y

x:y = -2:3 or x:y = 3:2.

So solution is x = 3 and y = 2
