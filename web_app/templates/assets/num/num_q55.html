Question :55

In a two digit number, the unit digit is equal to the square of tens digit. The difference between the original number and the number formed by interchanging the digits is 54. What is the original number

A.  24

B.  39

C.  54

D.  93

Answer: B

from the problem the difference between the original number and the number formed by interchanging the digits is 54.

take option a.

original number =24

reversing the number we get = 42

their difference = 42-24 = 18 not equal to 54.

take option b

original number = 39

reversing the number we get = 93

their difference = 93-39 = 54

So this is the original number
