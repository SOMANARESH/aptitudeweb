Question :51

The sum of the digits of a two digit number is 11. On adding 27 to the given number, its digits are reversed. Fid the number

A.  38

B.  47

C.  74

D.  83

Answer: B

Let the two digit number be XY then it can be represented in decimal format as below

(101×X)+(100×Y)--->1

By Reversing the digits of the number we get YX and it can be represented in decimal form as

(101×Y)+(100×X)--->2

Given that the sum of the digits of a number is 11

i.e. X+Y = 11--->3

If 27 is added to equation 1 then we get euqtion 2

i.e. (101×X)+(100×Y)+27 = (101×Y)+(100×X)

10X+Y +27 = 10Y+X

9Y-9X = 27

9(Y-X) = 27

Y-X = 3--->4

By solving eq.3 and eq.4 we will get

2Y = 14

Y = 7

From eq.4

Y-X = 3

X = Y-1

X = 7-3

X = 4

So the number is XY = 47
