A and B together have 158 , C has 101 less than what A and B together have and B has 23 more than C. Then amount of A is:

A.  80

B.  78

C.  57

D.  88

Answer: B

(A+B)=158

C has 101 less than what A and B together have

C = (A+B)−101=158−101

=57

B has 23 more than C

B = C + 23

B = 57+23

B = 80

Given A and B together have 158 A+B = 158

A+80 = 158

A = 158−80

A = 78
